App Overview  
The News Reader App is a React Native (Expo) application that shows technology news in real-time that is provided by another API.  
Users can:

View the latest articles  
Access the entire information on each article.  
Favorites: Save the favorite articles with the help of AsyncStorage.  
See favorites despite opening the app again.  
Use Bottom Tabs to navigate between Home, Favorites and Profile.  

Features

Home Screen
Retrieves news with News API.org.
Shows titles, descriptions and pictures through a FlatList.
Taps a detailed screen on an article.  

Detail Screen
Shows full article content  
Views big picture, description and content.  
Has button to Click on Full Article in browser.  

Favorites Screen
Stores favorite articles in the device with AsyncStorage.  
Saving articles is done when the app is re-opened.  
Allows removing favorites  

Profile Screen
Shows student information.  
Coverage- Assignment information and description of the app.  

API Used  
NewsAPI.org - Top Headlines API.
